# 云原生安全

## 目录结构
```
06_云原生安全/
├── 01_基础设施安全/
│   ├── 01_容器安全.md
│   ├── 02_集群安全.md
│   └── 03_网络安全.md
├── 02_应用安全/
│   ├── 01_镜像安全.md
│   ├── 02_运行时安全.md
│   └── 03_数据安全.md
└── 03_安全运维/
    ├── 01_安全监控.md
    ├── 02_漏洞管理.md
    └── 03_应急响应.md
```

## 说明

### 1. 基础设施安全
- 容器安全：容器运行时安全、容器隔离、权限控制
- 集群安全：API Server安全、etcd安全、节点安全
- 网络安全：网络隔离、流量加密、访问控制

### 2. 应用安全
- 镜像安全：镜像扫描、签名验证、仓库安全
- 运行时安全：进程控制、系统调用、资源隔离
- 数据安全：数据加密、访问控制、备份恢复

### 3. 安全运维
- 安全监控：指标监控、日志审计、告警管理
- 漏洞管理：漏洞扫描、风险评估、补丁管理
- 应急响应：事件处理、分析定位、恢复流程
